# Especificación de Requisitos de Software (SRS)

Este directorio contiene los documentos que conforman el Elemento de Configuración del Software (ECS) correspondiente a la *Especificación de Requisitos de Software* del proyecto Kontably. Estos documentos describen de manera formal y estructurada los requisitos funcionales y no funcionales del sistema, y constituyen la base para las fases posteriores de diseño, implementación y validación.
